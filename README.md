# doge-marketplace-docs

**Steps:**

1. Load projects (Overview)
2. List for sale / Create an offer
3. Fetch listings / offers for a project
4. Validate that the offer is still valid
5. Fetch activities
6. Load offer data
7. Load inscription content
8. Purchase
